# Phonebook App using socket
Phonebook Socket

## Files structure
```
-----phonebook-socket
    |------ client_data/        (data folder for client)
    |------ server_data/        (data folder for server)
    |------ config.yaml         (configuration yaml file)
    |------ client.py           (run client.py to create a connection to server)
    |------ client_utils.py     (send/recieve support function for client)
    |------ server.py           (run server.py to create a server)
    |------ server_utils.py     (send/recieve support function for server)
    |------ utils.py            (support functions)
    |------ read.py             (an inline configuration and database)

    # not important
    |------ create_data.py      (generate database)
    |------ object.py           (support create_data.py)
    
```
